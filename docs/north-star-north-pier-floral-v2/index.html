Studio process snapshot

How one Harbor Table arrangement is built in the studio.

Every arrangement starts on the same worktable with stem prep, branch structure, then vessel balancing so pickup and event installs hold shape in coastal air.

  1. Step 01

    Stem sorting and hydration

    Stems are sorted by bend strength, then staged in sea-salt buckets for conditioning.

  2. Step 02

    Branch frame and vessel lock

    Branch forms anchor negative space so movement reads natural from every table angle.

  3. Step 03

    Wrap, note card, and release check

    Final trim, paper wrap, and pickup timing check are completed before release windows open.
