# Home Hero Finalist Normalization Pass v2

Scope:
- Focused pass on existing finalists only.
- No new directions.
- No new matrix.
- No production UI changes.

## What changed in normalization

- Standardized all finalist pages to read as true full-screen above-the-fold hero concepts.
- Increased viewport occupancy for headline + anchor in each finalist.
- Reduced accidental dead space and removed small-module-in-large-canvas presentation.
- Strengthened typography scale and hierarchy consistency.
- Made CTA and proof placement intentional and readable in each concept.
- Preserved each finalist's core identity (composition family + anchor family).

## Finalists improved most

1. **C5B** (`c5b-floating-stage-object-theater.html`)
   - Biggest gain in stage scale and hero authority.
   - Anchor now reads as dominant launch object, not a floating widget.

2. **C4B** (`c4b-editorial-object-ledger.html`)
   - Editorial asymmetry now uses the full viewport convincingly.
   - Ledger rail + object stage relationship is clearer and stronger.

3. **C8B** (`c8b-framed-object-gallery.html`)
   - Framed gallery now feels like a complete hero composition.
   - Anchor cluster is large enough for fair visual judgment.

4. **C9E** (`c9e-productfirst-architectural-signature.html`)
   - Product-first dominance is now unambiguous.
   - Copy/proof no longer compete with the object environment.

## Finalists still weaker after normalization

1. **C7D** (`c7d-poster-luminous-campaign.html`)
   - Strong presence now, but still more campaign-expressive than broadly product-stable.
   - Slightly higher risk for tone drift versus top enterprise-ready directions.

2. **C3C** (`c3c-immersive-material-storyframe.html`)
   - Visually convincing at fullscreen, but cinematic mood can overpower utilitarian product cues depending on brand intent.

## Top 5 after normalization

1. **C5B** — Floating Stage Object Theater  
2. **C9E** — Product-first Architectural Signature  
3. **C8E** — Framed Architectural Spotlight  
4. **C4B** — Editorial Object Ledger  
5. **C5E** — Floating Stage Architectural Atrium  

Why this top 5:
- Best viewport usage and full-screen compositional confidence.
- Strong hero hierarchy and readable conversion path.
- Largest and most evaluable visual anchors.
- Strong commercial/premium presence without admin/dashboard regression.

## Best 3 candidates for full homepage development

1. **C5B** (`c5b-floating-stage-object-theater.html`)
2. **C9E** (`c9e-productfirst-architectural-signature.html`)
3. **C8E** (`c8e-framed-architectural-spotlight.html`)

Why these 3:
- They combine premium signal, product relevance, and hero clarity most consistently.
- Their presentation now supports fair real-world homepage evaluation.
- They remain visibly distinct from each other (stage, product-first, framed spotlight), giving strategic choice without repetition.
