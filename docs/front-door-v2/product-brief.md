# AI Web Factory Front Door v2 — Product Brief (Focused MVP)

## What AI Web Factory is
AI Web Factory is a guided AI launch-kit studio. It turns one structured business brief into a commercial launch package that an agent can execute, not just a generic website prompt.

## Target user
- Primary: founder, operator, or business owner with a concrete offer but limited launch-system clarity.
- Secondary: marketing lead who needs faster launch planning with clearer structure, proof notes, and production handoff.

## Core value proposition
From one guided intake, AI Web Factory produces:
- website direction
- conversion structure
- launch copy plan
- campaign angles
- proof and QA notes
- an agent-ready next-step production prompt

This reduces strategy ambiguity and avoids the "blank prompt" trap.

## Why this is not just website generation
Prompt-only website builders usually stop at page sections or visual output.

Front Door v2 positions AI Web Factory as a launch-brief engine that:
- captures business context before design generation
- selects a website archetype based on conversion and trust needs
- plans launch assets beyond pages (copy + campaigns + QA)
- creates an explicit agent handoff block for production

## MVP scope in this prototype
- Customer-facing homepage with clear positioning and launch-brief engine visual anchor.
- Guided business brief intake prototype with completeness scoring and live summary.
- Structured Launch Brief preview that demonstrates what downstream agents receive.
- Product docs defining architecture, schema, and self-evaluation.

## Intentionally not included yet
- Production UI integration into `app/` routes.
- Real backend persistence, authentication, or submission pipeline.
- Pricing, checkout, billing plans, or package calculator.
- Full multi-page marketing site expansion.
- Examples gallery or template marketplace.
- Automation hooks for external delivery channels.
